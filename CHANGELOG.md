# Installed
###  `GitHub Pages`
- `installed`: npm install gh-pages --save-dev
- `usage`: npm run deploy || npm run deploy -- -m "Deploy React app to GitHub Pages" 
- `link`: https://github.com/gitname/react-gh-pages

###  `Routes`
